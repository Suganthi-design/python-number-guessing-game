# python-number-guessing-game
I have just built my first fully functional Number Guessing Game.
Let's practice breaking problems into steps, writing real logic, and having fun while learning.
From taking the player's input, checking it against the correct number, giving hints, and generating a new random number.

Overview of Number Guessing Game

Here is how the game will work.

At the start, the computer secretly picks a number. The player's goal is to guess it. 🎲

If the guess is wrong, the game will:

Tell you the guess is incorrect.
Give you a hint to guess higher or lower.
Let you try again. 🔄
This continues until the player guesses the number correctly, at which point the game will display a congratulatory message.


Take Away Skills

Variables:	Storing and updating values like guessed_number and right_number.

Loops:	Letting the game keep running until the player guesses correctly.

Conditionals:	Checking the player's guess and deciding what to do next.

Modules:	Using Python's random module to make the game unpredictable.

User Experience:	Giving helpful hints and keeping the player engaged.
